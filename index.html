EO-SAR Binary Change Detection for Building Damage Assessment

Lightweight U-Net (ResNet34 backbone, 24.3M params) for pixel-level building damage detection on pre-event optical and post-event SAR image pairs. Built SAR preprocessing pipeline — Lee speckle filter, percentile normalization (p2–p98), zero-pixel masking — quantifying 20% cross-scene intensity domain shift. Tackled 1.57% class imbalance with Focal + Dice loss, boosting precision from 4% to 61%. Val F1 = 0.6083, IoU = 0.4371 with 4-variant TTA on 1024×1024 images.
